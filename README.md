🌲 Forest Guardian

A 3D interactive browser game where the player becomes the protector of an ancient forest.
Complete missions, collect spirit energy, and awaken the power of the forest to become the Forest Guardian.

The game is built using Three.js and runs entirely in the browser using WebGL, creating an immersive 3D forest environment with interactive missions.

🎮 Gameplay Overview

In Forest Guardian, the player explores a magical forest and completes three missions to unlock hidden powers.

Mission Flow

The Moss Shrine
Locate the ancient shrine and commune with the forest.

The Spirit Orbs
Walk through the forest and collect three floating spirit orbs.

The Heart Tree
Race through rising fog to reach the powerful Heart Tree and awaken the forest.

Each mission unlocks new abilities and upgrades the player's power.

✨ Features

🌳 Procedurally generated 3D forest environment

🎮 Interactive missions and objectives

🧭 Minimap navigation system

⏱ Timed challenges

⚡ Power-up progression system

🔥 Dynamic particle effects (fireflies, spores, aura)

🌫 Environmental effects like fog and lighting

🧠 Mission tracking UI and HUD

💡 Real-time 3D rendering using WebGL

🧩 Game Mechanics

Mission System – Sequential tasks that unlock new abilities.

Power Progression – Player becomes stronger after each mission.

Environmental Interaction – Shrines, orbs, and trees respond to the player.

Dynamic Camera System – Third-person perspective with smooth rotation.

Particle Systems – Fireflies, aura effects, and energy bursts.
